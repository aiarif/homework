HTML TAG

The tag represents the root of an HTML document. The tag is the container for all other HTML elements (except for the doctype tag).


Body Tag

The tag defines the document's body. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Comments Tag

The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.

Image tag

The tag is used to embed an image in an HTML page.
